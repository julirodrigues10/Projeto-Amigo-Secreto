 Projeto Amigo Secreto

Projeto desenvolvido em Java para realizar o sorteio de amigo secreto de forma simples e automática.

A aplicação permite cadastrar participantes e realizar o sorteio garantindo que ninguém tire a si mesmo.

Tecnologias utilizadas
- Java
- Lógica de programação
- Estruturas de dados (listas)
- Maven (se aplicável)

Funcionalidades
- Cadastrar participantes
- Listar participantes cadastrados
- Realizar o sorteio de amigo secreto
- Garantir que o participante não seja sorteado para si mesmo
